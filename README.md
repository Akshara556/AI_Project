# Persona-Adaptive Customer Support Agent

## Overview
This system dynamically adapts its response tone and content based on detected customer persona.

## Features
- Persona Detection (Rule-Based)
- Knowledge Base Retrieval
- Escalation Handling
- Context Handoff
- Streamlit Web Interface

## How to Run

### Backend (Terminal)
python app.py

### Web Interface
streamlit run streamlit_app.py
